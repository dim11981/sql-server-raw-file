## 0.0.1 (Jan 5, 2016)

Features:

  - added support of SQL Server Dts Raw File versions:
    * 00.90.00.00 => 9 (SQL Server: 9/10)
    * 00.10.01.00 => 10 (SQL Server: 11)
  - added support of basic data types:
    * int (i.e. NULL w/o cast)
    * float
    * bit
    * bigint
    * (n)(var)char
    * decimal, numeric
    * datetime
    * datetime2 (only SQL Server 10/11)

Bugfixes:

  - initial release
